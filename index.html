COMPONENT 01

⚡ The CPU

The Brain of the Machine

Imagine the CPU as the city's Mayor — every decision, calculation, and instruction flows through this one authority. When you press a key, click the mouse, or run an app, the CPU processes millions of tiny instructions per second to make it happen.

Modern CPUs have multiple cores — like having several deputy mayors, each handling different tasks simultaneously.

The CPU (Central Processing Unit) executes instructions using the Fetch-Decode-Execute cycle. It fetches an instruction from RAM, decodes its binary opcode, and executes the operation using the ALU (Arithmetic Logic Unit) or FPU (Floating Point Unit).

Modern CPUs use pipelining, out-of-order execution, and branch prediction to maximize instruction throughput. Clock speeds (GHz) measure cycles per second, while IPC (Instructions Per Cycle) determines real-world performance.

COMPONENT 02

🧠 RAM

Your Computer's Short-Term Memory

Think of RAM as your office desk. When you're working, you lay out all the papers, folders, and tools you need right now. The bigger your desk, the more you can work on simultaneously.

When you close an application, those "papers" are swept off the desk — the data is gone. That's why RAM is called volatile memory.

RAM (Random Access Memory) provides the CPU with ultra-fast, byte-addressable working memory. Modern DDR5 modules operate at 4800–6400 MT/s with 64-bit bus widths, offering ~50 GB/s bandwidth — critical for feeding data-hungry CPUs.

The CPU's memory controller manages address mapping, refresh cycles (DRAM loses charge ~every 64ms), and arbitrates between multiple cores. Cache hierarchy (L1/L2/L3) bridges the speed gap between CPU registers and DRAM latency.

🚀 DDR5 @ 6400 MT/s 📐 Up to 256 GB ⏱️ ~10 ns latency
COMPONENT 03

🎨 The GPU

The Visual Artist of Your System

If the CPU is the mayor making high-level decisions, the GPU is a massive team of artists — thousands of them — all painting tiny parts of your screen simultaneously.

Every pixel you see? The GPU calculated its color. From YouTube videos to 3D games to AI image generation — the GPU does it all in parallel.

A GPU (Graphics Processing Unit) contains thousands of small shader cores optimized for parallel floating-point math. Unlike the CPU's few powerful cores, a GPU like the RTX 4090 has 16,384 CUDA cores — ideal for the same operation applied across millions of pixels.

Modern GPUs also excel at tensor operations, making them indispensable for deep learning. The rendering pipeline: vertex shader → rasterization → fragment shader → framebuffer output — all happening 60+ times per second.

Data is encoded in binary (0s and 1s). In an HDD, tiny magnetic regions are flipped to represent bits. In an SSD, electrical charges are trapped in floating-gate transistors in NAND flash cells.

An NVMe SSD communicates via the PCIe bus — the same high-speed lane your GPU uses — bypassing the slower SATA controller entirely.

COMPONENT 04

📦 Storage

Your Computer's Long-Term Memory

Think of storage as your computer's bookshelf. Everything is saved here — your photos, documents, operating system, and apps — and it all stays even when you turn the computer off.

SSDs are like shelves with instant-access drawers; HDDs are like old filing cabinets where you have to wait for the drawer to roll out.

NAND flash memory stores data using floating-gate MOSFETs. SLC (1 bit/cell) is fastest; MLC (2-bit), TLC (3-bit), and QLC (4-bit) trade endurance and speed for higher density. NVMe's PCIe 4.0 interface delivers 64 GB/s theoretical bandwidth.

The filesystem (NTFS, ext4, APFS) manages file allocation, journaling, and wear leveling. The NVMe controller's built-in FTL (Flash Translation Layer) handles block mapping, garbage collection, and bad block management.

💾 Up to 8 TB SSD 🚀 7 GB/s NVMe 🔋 No moving parts
COMPONENT 05

🌐 Network

The World's Nervous System

If your computer is a city, the network is the highway system connecting every city on Earth. Data travels as tiny packets — like parcels on trucks — switching routes through thousands of routers to reach you in milliseconds.

When you load a webpage, hundreds of these packets bounce around the globe and reassemble perfectly on your screen.

The TCP/IP stack abstracts networking into layers: Physical → Data Link → Network → Transport → Application. Each packet carries a header with source IP, destination IP, TTL, and checksum. TCP provides reliable delivery with handshaking and retransmission; UDP sacrifices reliability for speed (used in video calls, gaming).

Modern Ethernet (10GbE) uses differential signaling over twisted pairs. Wi-Fi 6E (802.11ax) operates in the 6 GHz band with OFDMA and 160 MHz channels for multi-gigabit speeds. BGP routes packets across ASes (Autonomous Systems) on the global internet backbone.

🌍 300,000 km/s light speed 📡 Wi-Fi 6E: 9.6 Gbps 🔗 40ms global ping
